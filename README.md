# PolygonBilliards-PhysicsEngine
게임프로그래밍입문_Project#3

Custom 2D physics engine using:
- Convex polygon rigid body
- SAT collision detection
- Impulse-based collision response (with rotation)
- Cue force simulation
- Pocket detection

## How to Run
python main.py
